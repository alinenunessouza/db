## How to install:
  - *pipenv install* 

## How to run:
  - Navigate to the `store/src/api` folder 
  - *uvicorn app.main:app --reload* 

## Funcionalidades:

- Classes:
Produto -> adicionar(id, nome, ..)
Pedido -> criar(List<Produto>), atualizar
Usuario -> comprar, vender

- View:
Estoque
